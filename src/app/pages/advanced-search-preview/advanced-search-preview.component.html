About

This component is used for showing multiple inputs with different type of data. For now it supports: String, Number, Date, Boolean and Selection options. On button click component executes event provided in Even Emitter.

HTML call

It is called as any normal component. It has 1 input and 1 output: [params] of data type:

UniversalInput
key: string;
display: string;
type: DataType;selectOptions?: SelectionOptions[];

SelectOptions
 value: boolean | null | string;
viewValue: string;
